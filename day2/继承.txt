继承：
	继承是一种  is-a的关系，java是单继承，父类有子类也有，
	子类不能继承父类的成员有：1.private修饰的2.构造方法不能继承


java中4种访问权限修饰符，有严格到宽松
	private（私有）：本类
	friendly（默认）：同一个包可以使用，
	protected（继承）：本包可以使用，出了包有继承关系的子类也可以使用
	public（共有）：任何地方都可以使用
能修饰什么：	类	属性	方法
private（私有）：false  true    true
friendly（默认）：true  true    true
protected（继承）：false true   true  
public（共有）：true    true    true  

java 中只要继承能够得到，就可以重写
构造方法能不能重载，能不能重写
重写（头不能根小 身体必须一样 尾巴不能更大）
	发生在有继承关系的子类当中
	访问权限修饰符 不能更严格
	返回值类型 方法名 形参 必须一样
	抛出异常不能更广泛
super
只能出现在子类的方法和构造方法中
调用构造方法时，只能出现在第一句
不能访问父类的私有成员

抽象类可以有实例方法，有构造方法，但不能创建对象，仅仅是给子类的构造方法提供一条通路（super）

final和finally有什么区别：
finally：无论是否发生异常都会执行的代码
final：最终的（修饰符）
类：不能被继承
属性：不能被更改
方法：不能被重写
